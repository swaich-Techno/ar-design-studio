AR DESIGN STUDIO - FINAL FULL APP

This folder contains the complete source code for GitHub and Vercel.

IMPORTANT:
- If you upload this ZIP to GitHub, GitHub will store it as a ZIP file. It will NOT automatically extract it for Vercel.
- For Vercel deployment, GitHub repo must contain the extracted files: app, components, lib, models, public, package.json, etc.
- Best upload method: extract this ZIP, open the extracted folder, select all files/folders inside it, and drag them to GitHub.
- Do not upload node_modules, .next, .env, .env.local.

Photo uploads:
- Product image upload is available in Business > Products > Add/Edit Product.
- Business logo and cover upload is available in Business > Profile.
- Campaign banner upload is available in Business > Campaigns.
- Cloudinary env variables are required for uploads.

Fair usage and plan limits:
- Starter: 10 products, image catalogue only, product/catalogue QR, no included AR models, 1 MB max image upload.
- Growth: 25 products, up to 5 AR/3D products, table/campaign QR, 2 MB max image upload.
- Premium: 50 products, up to 15 AR/3D products, advanced analytics/staff, 5 MB max image upload.
- Cloud storage and AR model hosting are included within fair usage limits. Heavy 3D models, high traffic, or extra storage may require an add-on.
- Super Admin can see storage, AR product count, and product usage progress bars.

Mobile install:
- Android/iOS PWA files are included: manifest.json, sw.js, icons, /install, /offline.

MongoDB stores:
- Businesses/companies, users, products, table QR, campaign QR, analytics, audit logs.
