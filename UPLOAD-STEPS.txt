Best GitHub upload method:
1. Open GitHub repo.
2. Click Add file.
3. Click Upload files.
4. Open this folder on your computer.
5. Select all files and folders inside this folder.
6. Drag them into GitHub upload area.
7. Wait until all files finish loading.
8. Commit changes.

If browser upload fails again:
Use GitHub Desktop. It is better for full apps.

GitHub Desktop method:
1. Install GitHub Desktop.
2. Sign in with your GitHub account.
3. Clone repo: swaich-Techno/ar-design-studio.
4. Open the cloned repo folder.
5. Delete old repo files except .git if visible.
6. Copy all files and folders from UPLOAD-THIS-AR-DESIGN-STUDIO into the cloned repo.
7. In GitHub Desktop, write commit message: Upload AR Design Studio app.
8. Click Commit to main.
9. Click Push origin.

Vercel:
After GitHub upload, import the repo into Vercel and add environment variables from .env.example.
